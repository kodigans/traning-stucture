# Webpack config.
### Description:
This config allow handle files such as *.less, *.svg. To make one *.css file from many *.less files.

# Start application:

#### 1. Install dependencies:
```sh
$ npm install
```

#### 2. Build application:
```sh
$ npm run dev
```
